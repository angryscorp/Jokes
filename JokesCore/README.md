# JokesCore

## This package contains the main functionality of the `Jokes` app.

### The package has the following targets:

* `Common`
* `RootPresentation`
* Node `RandomJokes`
    * `RandomJokesData`
    * `RandomJokesDomain`
    * `RandomJokesInteracting`
    * `RandomJokesPresentation`
* Node `MyJokes`
    * `MyJokesData`
    * `MyJokesDomain`
    * `MyJokesInteracting`
    * `MyJokesPresentation`
* Node `NewJoke`
    * `NewJokeInteracting`
    * `NewJokePresentation`
* Node `Settings`    
    * `SettingsData`
    * `SettingsDomain`
    * `SettingsInteracting`
    * `SettingsPresentation`
